VoiceMax 1.0.0 — Quick Start
============================

Что это:
  Менюбар-приложение для macOS, которое:
  - расшифровывает речь с микрофона по нажатию fn (push-to-talk)
  - записывает встречи: твой мик + системное аудио вместе, с
    разделением на [Me] / [Others], в Markdown-файл

Требования:
  - macOS 13 (Ventura) или новее
  - Xcode Command Line Tools (один раз):
      xcode-select --install
  - Homebrew (опционально, нужен только для Telegram-фич):
      https://brew.sh

Установка:
  1. Распакуй этот архив куда хочешь, например в ~/voicemax/.
  2. Открой Terminal в этой папке и запусти:
       bash scripts/install.sh
     Скрипт сам соберёт VoiceMax.app, скопирует в /Applications/
     и запустит. Первая сборка занимает 1-2 минуты.

  3. macOS попросит разрешения по очереди:
     - Микрофон — обязательно
     - Accessibility (для глобальной fn-клавиши) — обязательно
     - Screen Recording — нужно только для режима встреч
       (захват системного аудио)

  4. Открой меню в трее (иконка справа в menu bar) и нажми
     "Deepgram API key…". Вставь свой ключ от Deepgram
     (получить: https://deepgram.com — есть бесплатный тариф).

Использование:
  - Push-to-talk диктовка: нажми fn, говори, нажми fn снова.
    Текст вставится в активное окно.

  - Запись встречи: в меню → "Start meeting".
    Файл появится в ~/Documents/VoiceMax-Recordings/.
    По умолчанию пишутся только собеседники из колонок/наушников.
    Чтобы твоя речь попала в транскрипт — зажми fn пока говоришь.

  - Просмотр транскрипта в реальном времени:
      tail -f "$(ls -t ~/Documents/VoiceMax-Recordings/*.md | head -1)"

Логи и диагностика:
    tail -f ~/Library/Application\ Support/VoiceMax/voicemax-tray.txt

Удалить:
    rm -rf /Applications/VoiceMax.app
    rm -rf ~/Library/Application\ Support/VoiceMax
    rm -rf ~/Documents/VoiceMax-Recordings

Подробнее в README.md.
